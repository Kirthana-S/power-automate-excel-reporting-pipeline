# Setup Guide

## Prerequisites

- Microsoft 365
- Power Automate
- SharePoint Online
- Excel Online
- Outlook

## Required Components

- SharePoint Document Library
- Office Script #1
- Office Script #2
- Outlook Connection

## High-Level Steps

1. Create a Power Automate flow.
2. Configure the Outlook trigger.
3. Save incoming attachments to SharePoint.
4. Execute Office Script #1.
5. Execute Office Script #2.
6. Save the processed report.
7. Send an email with the report attached.

> This guide is intentionally high level to protect confidential implementation details.
